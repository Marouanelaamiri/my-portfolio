Cub3D Fake 3D Game

Cub3D is a simple 3D game engine project inspired by the classic game Wolfenstein 3D. It is developed using the C programming language and utilizes the MiniLibX graphics library for rendering. The project focuses on implementing raycasting techniques to create a 3D environment from a 2D map, allowing players to navigate through a maze-like structure. Cub3D showcases fundamental concepts of computer graphics, game development, and low-level programming.
